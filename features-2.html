🛠️ الميزات الناقصة من متجر "Ahmed 🦅⚡ Store" (11 - 20)

11. تصنيف المنتجات بفئات فرعية Frontend

أمازون لا يكتفي بفئات عامة، بل يتيح تصنيفات فرعية (مثلاً: إلكترونيات ← سماعات ← بلوتوث).

كيف تضيفها؟ استخدم قوائم متداخلة في HTML/CSS مع روابط لتصفية حسب الفئة.

12. عرض العد التنازلي للخصومات Frontend

أمازون يستخدم مؤقتات للعد التنازلي للعروض لجذب الزوار. متجرك لا يحتوي على هذه الأداة.

كيف تضيفها؟ استخدم JavaScript لعرض Timer ينتهي بتاريخ معين.

13. الدفع الإلكتروني المباشر Backend

أمازون يسمح بالدفع عبر بطاقات الائتمان، PayPal، وغيرهم. متجرك يعتمد على نموذج يدوي فقط.

كيف تضيفها؟ استخدم بوابة دفع مثل Stripe أو Paymob لمصر، مع التكامل عبر API.

14. سجل الطلبات السابق Backend

أمازون يسمح للمستخدم بمراجعة طلباته السابقة. متجرك لا يحتفظ بأي سجل.

كيف تضيفها؟ استخدم قاعدة بيانات لربط الطلبات بحساب كل مستخدم.

15. حالة المنتج (متوفر / غير متوفر) Frontend

أمازون يعرض بوضوح هل المنتج متاح أو نفد من المخزون.

كيف تضيفها؟ أضف خاصية data-status لكل منتج تظهر في واجهة المستخدم.

16. شريط عرض سريع "منتجات تم مشاهدتها" Frontend

أمازون يعرض منتجات زارها المستخدم سابقًا. متجرك لا يخزن زيارات.

كيف تضيفها؟ استخدم localStorage لحفظ آخر المنتجات التي زارها المستخدم.

17. حماية ضد الطلبات المتكررة Backend

أمازون يمنع إرسال نفس الطلب مرتين. متجرك لا يملك حماية ضد التكرار.

كيف تضيفها؟ تحقق من البريد / رقم الهاتف قبل إرسال النموذج مرة ثانية.

18. أسعار مخفضة تلقائيًا عند الكمية Backend

أمازون يقدم خصم تلقائي عند شراء أكثر من قطعة.

كيف تضيفها؟ استخدم JavaScript لحساب خصم بناءً على الكمية في زر "الشراء".

19. دعم تعدد العملات Frontend

أمازون يسمح بعرض الأسعار بالدولار أو عملة محلية. متجرك يعرض فقط بالجنيه.

كيف تضيفها؟ استخدم API لتحويل العملات وعرضها باستخدام select لتغيير العملة.

20. صندوق أسئلة وأجوبة (Q&A) Frontend + Backend

أمازون يسمح للمستخدمين بطرح أسئلة حول المنتج والبائع يرد عليها.

كيف تضيفها؟ نموذج إرسال سؤال + قاعدة بيانات أو Google Sheets لحفظ الأسئلة والأجوبة.
