🛠️ الميزات الناقصة من متجر "Ahmed 🦅⚡ Store" (61 - 70)

61. دعم البيع من عدة بائعين (Marketplace) Backend

أمازون يسمح لأي شخص بالبيع على المنصة. متجرك حاليًا أحادي البائع.

كيف تضيفها؟ أنشئ لوحة تحكم للبائعين لإضافة منتجاتهم بعد التسجيل والموافقة.

62. صفحة "منتجات مشابهة" تلقائيًا Frontend

عند دخول أي منتج، أمازون يعرض اقتراحات مشابهة.

كيف تضيفها؟ أضف قسمًا أسفل المنتج يستخدم نفس الفئة أو الوسوم لعرض منتجات مشابهة.

63. التحكم في ترتيب المنتجات (الأحدث - السعر - الأكثر تقييمًا) Frontend

أمازون يسمح بترتيب النتائج حسب معايير مختلفة.

كيف تضيفها؟ أضف قائمة <select> لتحديد طريقة الترتيب، واستخدم JavaScript لتحديث العرض.

64. إدخال عنوان الشحن على خريطة Frontend + Backend

أمازون يسمح باستخدام الموقع الجغرافي لتحديد العنوان بدقة.

كيف تضيفها؟ استخدم خرائط Google Maps API في صفحة الطلب، لحفظ الموقع كنقطة GPS.

65. تقييم سرعة التوصيل لكل منتج Backend

أمازون يقدر توصيل كل منتج حسب البائع أو المدينة.

كيف تضيفها؟ أضف حقل "مدة التوصيل" لكل منتج (3 أيام، 7 أيام...)

66. زر “اشترِ لاحقًا” Frontend

أمازون يتيح حفظ المنتج في قائمة “لاحقًا”.

كيف تضيفها؟ أضف زرًا يحفظ ID المنتج في localStorage أو قاعدة بيانات المفضلة.

67. عرض الكمية المتاحة مباشر من المخزون Backend

أمازون يُظهر عدد القطع المتاحة (مثلاً: تبقى 4 قطع فقط!).

كيف تضيفها؟ أنشئ لكل منتج قيمة stock تُنقص تلقائيًا بعد كل طلب.

68. تنبيه عند وجود تقييمات سلبية Backend

أمازون ينبه البائع إذا حصل على تقييمات أقل من 3 نجوم.

كيف تضيفها؟ تحقق من متوسط التقييم وأرسل تنبيهًا داخليًا أو عبر بريد.

69. إمكانية تغيير العملة يدويًا Frontend

أمازون يعرض الأسعار بالعملة التي يختارها المستخدم.

كيف تضيفها؟ استخدم محول عملات API مع خيار يدوي في القائمة.

70. حساب الوزن الإجمالي للطلب Backend

أمازون يحسب الوزن الكلي للشحن.

كيف تضيفها؟ أضف قيمة وزن لكل منتج، واجمعها تلقائيًا عند ملء سلة الشراء.
